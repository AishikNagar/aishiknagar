Portfolio

Portfolio

Recent Projects

Sherlocked-VR

We built a Virtual Reality Escape room game as a part of the NUS Hack&Roll Hackathon. We used Unity engine, Google Virtual Reality unity SDK, C# and Android NDK to make this game.

PokeCenter

I created a chatroom for pokemon enthusiasts using Node.JS, Express, web sockets using sockets.io, HTML, CSS and bootstrap. It had seperate rooms for anime, manga, pokemon go and competitive gaming fanatics.

CartPole-AI

I built an AI agent which played the balance cart-on-the-pole game using tensorflow, keras and OpenAI gym. It was an intense learning experience as I learnt to implement policy gradients to build a deep renforcement learning agent.

Stylize

I built a flask web app for everything image related. I used transfer learning and Generative Adversarial Networks to add a style transfer functionality where the user can input content image and style image of their choice and recieve a style transferred output. I also used Caman.js to add photoshop functionality and used detecto library for object detection functionality.

Edu-Pic

This project was made as a part of the EnlighteNUS hackathon in which we secured a position in the Top 7. We used flask to build a web-app which lets user capture an image and gives the user the science behind the object. We used OpenCV and Google Cloud Vision API in order to capture images and objects in them. We scraped the web using beautiful soup in order to obtain information about the object and relay it to the user.

Pong Browser AI

I used Tensorflow.js, HTML,CSS and JavaScript to create an AI which learns to play pong after watching the user. Using tesorflow.js, I was able to run the machine learning model on the client side in the browser.

DevSite

I created a social network for developers using Node.js, Express, React.JS, Redux and MongoDB. This full stack project included login and authentication using JSON web tokens, avatar management using Gravatar, React Hooks and was deployed on heroku with a postbuild script.

Craigslist Clone

I made a clone of the craigslist website using Django, MySQl, python, HTML, CSS and Javascript. This site provided a clean UI to search items on craigslist. I used Beautiful Soup to scrape data,prices,images and descriptions from craigslist and redirect user to the actual site.
